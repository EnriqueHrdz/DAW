3.¿Qué debería realizar si deseo eliminar los elementos de la lista al realizar un click al listado?

El este caso no se puede por que no se realiza ninguna función para eliminar.

Pero si se desea:

function eliminar(elemento)
        {
            var id=elemento.parentNode.getAttribute("id");
            node=document.getElementById(id);
            node.parentNode.removeChild(node);
        }
